# React Template

A minimal React + Webpack boilerplate with linting and formatting preconfigured.

---

## 🚀 Getting Started

### Setup

```bash
git clone --depth 1 https://github.com/Ashish-Dhami/template-react.git my-app
cd my-app
npm run setup
```

---

## 📦 What this does

- Clones only the latest snapshot (`--depth 1`) for faster setup
- Removes template git history and initializes a fresh repository
- Installs:
  - React dependencies
  - Development tools (Webpack, ESLint, Prettier, Stylelint)

---

## 🛠 Available Scripts

```bash
npm run start      # Start development server
npm run build      # Build for production

npm run lint       # Run all linters
npm run format     # Format code

npm run clean      # Remove dist and node_modules
```
